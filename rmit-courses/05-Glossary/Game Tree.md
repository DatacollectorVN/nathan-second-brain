## Definition
> A search tree representing a game: nodes are game states, edges are moves, alternating levels correspond to the players' turns, and leaves are terminal states with utility values.

## Context
> AI Lecture 03: the full tree is impractical even for tic-tac-toe, so search runs from the current state to a depth/cutoff instead.
