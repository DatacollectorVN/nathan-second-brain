## Definition
> A game where players' utilities at any terminal state sum to a constant — one player's gain is the other's loss. Chess uses terminal utilities −1, 0, or 1.

## Context
> Introduced in AI Lecture 03 when formally defining Utility(s, p); the setting minimax assumes.
