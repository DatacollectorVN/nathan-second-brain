title: Quiescent State
tags: [glossary]
---

## Definition
> A game state that is "calm" — nothing dramatic (like a piece capture) is imminent — so a heuristic evaluation function can be applied reliably.

## Context
> AI Lecture 03: apply Eval(s) only to quiescent states (not "I am about to lose my queen!"); non-quiescent states should be expanded further.
