## Definition
> A pattern of data describing how frequently different values of a variable occur, from which random samples can be drawn; always characterized by parameters (averages, standard deviations, rates, etc.).

## Intuition
> The course introduces it as a "frequency distribution" — e.g., a histogram of frequencies per month. If you know the shape and parameters of how an input behaves, you can generate as many plausible values of it as you like instead of guessing one.

## How It Works
In [[Monte Carlo Simulation]], each input variable is assigned the distribution that best represents its behavior (the course's airline-catering case used Normal, Uniform, and Binomial fits), its parameters are estimated, and a random number generator draws N values from it per simulation run.
