## Definition
**ALFWorld** (Shridhar et al., 2020) — a synthetic **text-based game** aligned with the embodied ALFRED benchmark, where an agent achieves household goals (e.g. "put a pepper shaker on a drawer") via text actions over long horizons with sparse rewards. 6 task types.

## Context
An interactive decision-making benchmark in [[ReAct - Synergizing Reasoning and Acting in Language Models|ReAct]], evaluated on **134 unseen games**. Best ReAct reaches **71%** success vs the imitation-learning baseline BUTLER's 37% — an absolute +34%.
